# Eto-Sur
My 1st 3D game in Unity engine.

This is 2 players game. You need to dodge bombs, lasers, freeze zones and collect bonus health
thereby leveling up and get bonus movespeed, jumpforce.

Controls:
- 1st player:
  - WASD - moving;
  - J - jump.
- 2nd player:
  - Arrows - mooving;
  - Enter - jump.
